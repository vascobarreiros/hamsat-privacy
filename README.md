# hamsat-privacy
HamSat — Privacy Policy
Last updated: April 2026

HamSat does not collect, store, or transmit any personal data.

Location: Used on-device only to calculate satellite passes. 
Never sent to any server.

Analytics: None. No tracking or advertising SDKs.

Data sources: Publicly available orbital data from CelesTrak, 
AMSAT, and N2YO. No personal information is sent to these services.

Purchases: Handled entirely by Apple. We never see your 
payment information.

Contact: vasco@martynet.pt
